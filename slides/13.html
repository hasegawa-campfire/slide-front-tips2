CSS Nesting Module

Sassのようにネストの形で定義する

[Editor's Draft] CSS Nesting Moduleは、Sassのようにネストする形でCSSが書けるようになる仕様です。 これは欲しいですね。 私自身、これがCSSに無いからという理由でSassを使っているような気もします。 変数は既にCSSにもありますし、それを使って動的に色を変換できる次のCSS Color Moduleも将来的には控えています。 mixin的なCSSの仕様はもう既に消えてしまいましたが、それでも素のCSSが十分選択肢になりえる、そういう未来もあるのかもしれません。 楽しみですね。
