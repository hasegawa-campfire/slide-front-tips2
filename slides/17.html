画像検出や音声認識など

$video.srcObject = await navigator.mediaDevices.getUserMedia({ video: true }) const faceDetector = new FaceDetector() const faces = await faceDetector.detect($video) console.log(faces[0].boundingBox) const recognition = new SpeechRecognition() recognition.onresult = (e) => console.log(e.results[0][0].transcript) recognition.start()
オマケです。画像検出や音声認識はブラウザ上でも出来ます、という紹介です。 カメラの映像を取り込んで、顔を検出。そして音声認識。それぞれ数行で簡単に利用できます。 とは言え、現状はかなり限定された環境でしか動きません。なのでオマケです。 ちなみに画像検出は、テキストやQRコードの検出もあるようです。顔よりは幾分、実用的な気がしますね。
